# 🌾 AgriAdvisor

AgriAdvisor is a Data Science and Machine Learning based web application that helps farmers make better agricultural decisions.

## 🚀 Features

- 🌱 Crop Yield Prediction
- 🧪 Fertilizer Recommendation
- 📊 User-friendly Streamlit Interface
- 🤖 Machine Learning Model (Random Forest)
- 📈 Data Preprocessing and Model Training

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## 📂 Project Structure

```
AgriAdvisor/
│── app.py
│── model.pkl
│── dataset.csv
│── requirements.txt
│── README.md
│── images/
└── notebooks/
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/kailashchaudhary-p/AgriAdvisor.git
```

2. Go to the project folder

```bash
cd AgriAdvisor
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
